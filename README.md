# WEBSITE-PERSONAL-ATHIFAH
Proyek ini adalah tugas mata kuliah **Pemrograman Web** di semester 2.
# Deskripsi Proyek
Website ini merupakan profil personal statis yang mencakup:
* **Halaman Home**: Profil singkat dari program studi
* **Curriculum Vitae**: Berisi identitas diri, riwayat pendidikan, kelebihan
* **Hobby**: Berisi kegemaran dan daftar buku yang telah dibaca

Anda bisa melihat hasilnya disini:
